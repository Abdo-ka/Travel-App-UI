
# Travel App

this is just UI for simple Travel app 
 
## Run Locally

Clone the project

```bash
  git clone https://github.com/Abdo-ka/Travel-App-UI.git
```

Go to the project directory

```bash
  cd travel_app
```

Install dependencies 

```bash 
  flutter pub get
```

Install dependencies on packages 

```bash 
  cd packages
  flutter pub get 
```
## Build Application

release version
```bash
flutter build apk --release
```
